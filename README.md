# Desk-Edge Power Strip Rail System

A modular, non-permanent mounting solution for power strips, designed for portability and stability without the use of adhesives or screws.

## Project Overview
The goal of this project was to create a mounting system for a power strip that can be quickly attached, detached, or moved along the edge of a desk. 

### Key Features:
* **Friction-Fit Clips:** Designed to grip the desk edge securely using a cantilever-style profile.
* **Modular Rail Design:** The main holder slides into the clips, allowing for easy removal of the power strip while keeping the mounts in place.
* **Ventilated Geometry:** The holder features a diagonal slot pattern to reduce material usage and allow for heat dissipation from power adapters.

## Technical Specifications
* **Software:** Autodesk Inventor.
* **Validation:** Performed a Von Mises Stress Analysis (FEA) to ensure structural integrity.
    * **Max Stress Observed:** 0.1645 MPa.
    * **Result:** High Factor of Safety (FoS) for standard PLA material.

## Manufacturing Details
The parts were manufactured using Fused Deposition Modeling (FDM).
* **Printer:** Bambu Lab P1S.
* **Material:** PLA Basic.
* **Slicer Settings:**
    * **Infill:** 15% (Grid pattern).
    * **Wall Loops:** Standard.
    * **Layer Height:** 0.20mm.
